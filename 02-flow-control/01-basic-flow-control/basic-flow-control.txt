Flow Charts & Basic Flow Control Concepts
-----------------------------------------

Flow Chart
----------
- A metaphor for how the execution can move around your program
    - Control statements decide which Python instruction to execute under w/c condition(s)


Boolean Values
--------------
- Python's Boolean data type has only two values
    - True
    - False

Comparison Operators
--------------------
- These operators are used in expressions
    - Equal to                  ==
    - Not equal to              !=
    - Less than                 <
    - Greater than              >
    - Less than or Equal to     <=
    - Greater than or Equal to  >=

Boolean Operators
-----------------
- These operators are used in expressions
    - And   &&
    - Or    ||
    - Not   !

- And Truth Tables
    - True && True      = True
    - True && False     = False
    - False && True     = False
    - False && False    = False

- Or Truth Tables
    - True || True      = True
    - True || False     = True
    - False || True     = True
    - False || False    = False

- Not Truth Table
    - ! True            = False
    - ! False           = True
